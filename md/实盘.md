\*\*# 实盘分析系统 — 功能设计文档 (PRD)

> **数据来源**：仅 API 对接交易所
> **结算时间**：UTC 0 点
> **盈亏计算**：包含手续费（真实收益）
> **无风险利率**：稳定币活期利率（约 3-5%，可配置）
> **未平仓订单**：不计入当日收益，仅展示当前持仓

---

## 一、 API 数据对接层

### 1.1 API Key 管理

- [ ] **支持交易所列表**：Binance、OKX、Bybit、Bitget、Gate.io、MEXC（架构可扩展）
- [ ] **权限校验**：强制只读权限检测，若检测到写入权限直接拒绝绑定并提示用户
- [ ] **Key 加密存储**：AES-256 高强度加密，服务端严禁保存明文
- [ ] **命名与标签**：用户自定义账户名（如“主账户”、“网格测试账户”、“套利账户”）
- [ ] **编辑与删除**：支持更换 Key（保留历史数据关联）；删除 Key 后保留已同步历史数据
- [ ] **暂停同步**：支持临时停用同步，不删除 Key 配置
- [ ] **同步状态监控**：实时显示上次同步时间、同步结果、失败原因
- [ ] **手动触发同步**：提供用户主动刷新/补采数据按钮
- [ ] **多交易所合并**：
  - 按 USDT 本位合并计算总净值
  - 同名币种持仓跨交易所合并展示
  - **跨交易所套利/分仓处理**：汇总各交易所的名义敞口总和与已用保证金总和，反映真实资金占用与总体风险

### 1.2 数据同步机制

| 功能               | 说明                                                                                  |
| :----------------- | :------------------------------------------------------------------------------------ |
| **首次全量同步**   | 默认拉取全部历史，支持用户选择起始日期；分页拉取、进度展示、断点续传                  |
| **定时增量同步**   | 每 5 分钟自动同步（可配置）；基于`last_trade_id` 或时间戳增量获取；按 `trade_id` 去重 |
| **API 限流管理**   | 自动退避重试（指数退避算法），适配各交易所 Rate Limit                                 |
| **API 失效/过期**  | 异常检测，通过邮件/App 推送即时提醒用户更新 Key                                       |
| **交易所维护监控** | 自动识别交易所维护状态标记，恢复后自动补同步                                          |
| **数据缺口补偿**   | 定期检测时间缺口，自动触发断层补全拉取                                                |

### 1.3 数据标准化映射

| 维度             | 处理方式                                                                              |
| :--------------- | :------------------------------------------------------------------------------------ |
| **交易对标准化** | 统一映射为`BTC/USDT` 标准格式；精确区分现货/永续/交割/期权                            |
| **方向标准化**   | 现货：`BUY` / `SELL`；合约：`OPEN LONG` / `CLOSE LONG` / `OPEN SHORT` / `CLOSE SHORT` |
| **强平标记**     | 独立标记强平订单（Liquidation），强平订单不参与胜率计算                               |
| **手续费处理**   | 支持多币种手续费；按成交时汇率折算为 USDT；识别返佣/抵扣记录冲减成本                  |
| **时间标准化**   | 统一 UTC 存储；前端展示自动换算为用户时区；自动适配夏令时                             |

## 二、 资金曲线模块

> 展示账户净值随时间变化的趋势，支持手动与自动刷新。

### 2.1 曲线展示

- [ ] **净值走势**
  - Y 轴左侧：账户净值（USDT 本位）
  - Y 轴右侧：累计收益率（%）
  - X 轴：时间（日/周/月）
  - 支持多账户叠加对比，不同颜色区分
- [ ] **基准对比**
  - 支持叠加 BTC、ETH 及大盘指数走势
  - 显示相对基准的超额收益曲线（Alpha）
- [ ] **关键节点标记**
  - **充值/提现**：垂直虚线标记，Hover 显示金额与时间，净值曲线自动剔除出入金干扰
  - **最大回撤区间**：自动识别并高亮起止点，显示回撤幅度与持续天数
  - **自定义标记**：用户手动标记事件（如“切换策略”、“停盘”、“开启网格”），支持文字说明

### 2.2 时间范围

- [ ] **快捷切换**：7天 / 30天 / 90天 / 1年 / 本年 / 全部
- [ ] **自定义范围**：日期选择器，最长跨度无限制
- [ ] **缩放与平移**：鼠标滚轮缩放，拖拽平移，双击重置为默认范围

### 2.3 数据交互与性能

- [ ] **Hover 详情**：日期、当日净值、累计收益率、距前期高点回撤幅度
- [ ] **点击下钻**：点击某日数据点，直接跳转当日交易明细页
- [ ] **数据降采样**：大范围时自动降采样（LTTB 算法），保证渲染流畅度；小范围展示完整每日数据

### 2.4 刷新机制

| 方式         | 说明                                         |
| :----------- | :------------------------------------------- |
| **自动刷新** | 每 5 分钟自动拉取最新净值（可配置开关/间隔） |
| **手动刷新** | 刷新按钮，点击立即触发拉取                   |
| **刷新状态** | 显示上次刷新时间，刷新中显示 Loading         |
| **异常提示** | 刷新失败时提示具体原因，支持一键重试         |

## 四、 资金利用率与杠杆分析模块

> **重构说明**：针对加密货币市场高杠杆与双重账户（现货+合约）特性，将资金利用率拆解为**保证金占用**与**真实名义敞口（实际杠杆）**两个维度，彻底解决高倍杠杆下“保证金极小但风险极高”的识别难题。

### 4.1 双轨计算模型

| 指标维度                                  | 计算公式                                                                                    | 业务含义与用途                                                                                                   |
| :---------------------------------------- | :------------------------------------------------------------------------------------------ | :--------------------------------------------------------------------------------------------------------------- |
| **保证金利用率**<br>_(Margin Allocation)_ | $\frac{\text{现货市值} + \text{合约已用保证金} + \text{挂单冻结保证金}}{\text{总账户净值}}$ | **资金流转视角**：评估账户闲置资金量，判断是否有余钱用于质押理财或追加保证金。                                   |
| **实际总杠杆**<br>_(Effective Leverage)_  | $\frac{\text{现货市值} + \text{合约名义持仓价值 (Notional Value)}}{\text{总账户净值}}$      | **风险敞口视角**：评估真实风险敞口（如 $\$10,000$ 净值用 100x 开 $\$100,000$ 仓位，实际杠杆为 **$10\times$**）。 |
| **合约专项杠杆**                          | $\frac{\sum \|\text{合约单持仓数量} \times \text{标记价格}\|}{\text{合约账户净值}}$         | 隔离评估合约子账户自身的真实风险倍数。                                                                           |

### 4.2 数据采集与频率

| 维度     | 采集数据项                                                                                                          | 采集频率                                                                                          |
| :------- | :------------------------------------------------------------------------------------------------------------------ | :------------------------------------------------------------------------------------------------ |
| **现货** | 现货账户余额、标记价格、可用余额（`free`）                                                                          | **定时快照**：每 5 分钟<br>**交易触发**：成交、挂单、撤单后即时快照<br>**手动刷新**：用户主动刷新 |
| **合约** | 持仓保证金、未实现盈亏（`unrealizedProfit`）、挂单冻结保证金（`openOrderMarginFrozen`）、持仓名义价值（`Notional`） | ~                                                                                                 |
| **理财** | 质押/理财账户金额                                                                                                   | ~                                                                                                 |

### 4.3 展形式与仪表盘设计

- [ ] **实时仪表盘（双核心指标）**
  - **指标卡 1：资金占用状态 (Margin Usage)**
    - 展示保证金利用率（如 `15.5%`），附带可用于理财/追加保证金的空闲资金量。
  - **指标卡 2：真实风险敞口 (Effective Leverage)**
    - 展示实际总杠杆倍数（如 `3.45x`）及总名义敞口金额。
    - 字体颜色与预警等级：`0-1x`（灰色/低风险）、`1-3x`（绿色/稳健）、`3-5x`（黄色/中高风险）、`5-10x`（橙色/高风险）、`>10x`（深红闪烁/极危）。
  - **环形进度条 & 构成饼图**：展示现货市值、合约保证金、挂单冻结、未实现盈亏的占比分布。
- [ ] **利用率 vs 杠杆二维散点图（四象限分析）**
  - X 轴：保证金利用率 (%)
  - Y 轴：实际总杠杆 (x)
  - 划分为：【高杠杆·小资金占用（高爆仓风险）】、【高杠杆·重仓满杠（极度危险）】、【低杠杆·资金闲置】、【低杠杆·稳健持仓】四象限。
- [ ] **时段热力图**：24h × 7天矩阵，展示资金占用与杠杆使用的高峰时段。

### 4.4 预警机制

| 预警类型           | 触发条件                                                 | 通知方式               | 提示文案 / 建议                                                         |
| :----------------- | :------------------------------------------------------- | :--------------------- | :---------------------------------------------------------------------- |
| **高杠杆预警**     | 实际杠杆$> 5\times$ 持续 5 分钟                          | App 推送 + 邮件        | `当前账户实际杠杆已达 5.2x，风险敞口较大，请警惕剧烈波动！`             |
| **虚假安全预警**   | 保证金利用率$< 10\%$ 且 实际杠杆 $> 5\times$             | App 推送 + 邮件        | `名义敞口过大！虽然保证金占用仅 8%，但实际杠杆已达 5x 以上，谨防爆仓。` |
| **极限杠杆预警**   | 单个合约仓位实际杠杆$> 20\times$                         | 紧急推送 + 短信 + 声音 | `检测到高倍杠杆仓位（BTC/USDT），市场波动极易引发强平！`                |
| **资金闲置预警**   | 保证金利用率$< 10\%$ 且 实际杠杆 $< 0.5\times$ 持续 7 天 | App 内提醒             | `资金闲置过久，建议配置稳定币活期理财（预计年化 3-5%）。`               |
| **利用率骤降预警** | 单日内利用率从$>80\%$ 降至 $<30\%$                       | App 推送               | `仓位大幅变动，请确认是否为预期平仓/撤单操作。`                         |

### 4.5 关联分析

- [ ] **区间收益对比**：不同杠杆区间与保证金利用率区间的平均收益率与胜率对比。
- [ ] **最优仓位区间识别**：自动计算历史夏普比率最高时的杠杆与资金利用率区间。
- [ ] **高利用率回撤分析**：统计高利用率/高杠杆期间的平均回撤幅度与爆仓关联度。

## 五、 交易质量分析模块

### 5.1 胜率与盈亏分析

- [ ] **胜率多维统计**
  - **总体胜率**：全部历史成交数据
  - **趋势胜率**：按周/月/季度统计胜率变化趋势
  - **分币种胜率**：各交易对胜率对比柱状图
  - **分方向胜率**：做多胜率 vs 做空胜率（合约）
  - **连胜 / 连败**：记录最长连胜、最长连败笔数及当前连胜/连败状态
- [ ] **盈亏比分析**
  - 总体盈亏比、分币种盈亏比、分时段盈亏比趋势
  - **盈亏分布直方图**：所有交易笔数在不同盈亏金额区间的直方图
- [ ] **期望值分析**
  - 每笔交易数学期望值（$E = \text{胜率} \times \text{平均盈利} - \text{败率} \times \text{平均亏损}$）
  - **期望值滚动曲线**：近 30 笔交易期望值趋势
  - **期望值归零预警**：连续 20 笔滚动期望值 $< 0$ 时触发策略退化预警

### 5.2 交易频率与效率

| 指标                              | 说明                                                                                                                    |
| :-------------------------------- | :---------------------------------------------------------------------------------------------------------------------- |
| **日均交易笔数 / 天数占比**       | 总笔数 / 交易天数；有交易天数 / 总天数                                                                                  |
| **持仓时长分析**                  | 平均持仓时长；分布（$<1\text{h}$ / $1\text{-}4\text{h}$ / $4\text{-}24\text{h}$ / $1\text{-}7\text{d}$ / $>7\text{d}$） |
| **盈亏 vs 持仓时长散点图**        | 识别“拿不住盈利单”或“死扛亏损单”的心理问题                                                                              |
| **平均保证金占用 & 平均实际杠杆** | 评估交易过程中的平均资金效率与风险偏好                                                                                  |

## 六、 风险指标模块

### 6.1 回撤分析

- [ ] **回撤指标**
  - **历史最大回撤 (Max Drawdown)**：成立以来最大跌幅
  - **当前回撤 (Current Drawdown)**：当前净值距前期高点的跌幅
  - **回撤持续 / 恢复天数**：当前已持续回撤天数，历史最大回撤的恢复耗时
- [ ] **回撤可视化与归因**
  - **回撤深度图**：净值曲线下方填充回撤区域，渐变色显示
  - **回撤归因**：关联回撤期间的交易明细，识别导致主要回撤的“坏交易”
- [ ] **回撤预警**：用户自定义预警线（如 10%、20%），达到阈值即时推送建议

### 6.2 风险调整后收益

| 指标                           | 计算说明                                                     | 作用                                   |
| :----------------------------- | :----------------------------------------------------------- | :------------------------------------- |
| **夏普比率 (Sharpe Ratio)**    | $\frac{R_p - R_f}{\sigma_p}$（支持 30天/90天/1年滚动及年化） | 评估每承受一单位总风险所获得的超额收益 |
| **索提诺比率 (Sortino Ratio)** | $\frac{R_p - R_f}{\sigma_d}$（仅计算下行标准差 $\sigma_d$）  | 剔除向上波动干扰，更贴合交易真实风险   |
| **卡玛比率 (Calmar Ratio)**    | $\frac{\text{年化收益率}}{\text{历史最大回撤}}$              | 评估收益与最大风险的比值               |

## 七、 仓位分析模块

### 7.1 币种仓位分布

- [ ] **实时持仓**：币种、数量、市值、占比、未实现盈亏、收益率；支持饼图与 Treemap 树状图展示。
- [ ] **历史仓位回放**：时间轴滑块支持查看任意历史日期的仓位结构；对调仓占比 $>20\%$ 的日期自动高亮标记。

### 7.2 方向与杠杆分析（合约）

- [ ] **多空占比与净敞口**：做多总市值 vs 做空总市值，计算净敞口偏好（Long/Short Bias）。
- [ ] **杠杆与盈亏散点图**：实际杠杆倍数 vs 盈亏金额，识别最佳获利杠杆区间。

### 7.3 交易时段与周期性分析

- [ ] **24h × 7天 热力图**：展示交易频率、盈亏金额、胜率在各时段的分布。
- [ ] **熬夜交易识别**：统计 UTC 0-6 点（深夜时段）的交易占比与胜率，警惕疲劳交易。
- [ ] **周期效应**：月初/月末、季度初/末的胜率与盈亏对比。

## 八、 复盘记录模块

### 8.1 自动复盘草稿

- [ ] **每日自动摘要**：定时生成交易日报（笔数、盈亏、最大单笔、胜率、异常交易标记）。
- [ ] **关键交易强制复盘提醒**：
  - 单笔亏损 $> \text{净值 } 3\%$ 时提示填写复盘
  - 连续 3 笔亏损或处于最大回撤期时提醒复盘

### 8.2 手动复盘与标签系统

- [ ] **模版与富文本**：预设系统符合度、入场/出场依据、心理状态等模版；支持富文本、K线截图上传、语音转文字。
- [ ] **标签库管理**：
  - **情绪标签**：贪婪、恐惧、侥幸、后悔、冷静
  - **技术标签**：追涨、抄底、扛单、按计划止损
  - **系统标签**：完全符合系统、偏离系统
- [ ] **标签统计分析**：统计不同标签（如“恐惧”或“偏离系统”）下的平均胜率与盈亏比。

### 8.3 复盘回顾与导出

- [ ] **时间轴与日历视图**：高亮有复盘的日期，支持按标签、关键词检索。
- [ ] **导出功能**：支持导出 PDF 复盘报告或 Markdown 格式文件。

## 十、 数据存储与 retention 策略

| 数据类型         | 存储粒度                 | 保留周期 | 推荐存储引擎             |
| :--------------- | :----------------------- | :------- | :----------------------- |
| **原始快照**     | 5 分钟                   | 90 天    | TimescaleDB / ClickHouse |
| **小时汇总**     | 小时均值/最高/最低       | 1 年     | TimescaleDB / ClickHouse |
| **日/周/月汇总** | 日均值/区间分布/平仓明细 | 永久     | PostgreSQL / MySQL       |
